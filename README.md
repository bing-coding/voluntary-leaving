# 自愿离校

《自愿离校》是一款静态网页解谜游戏。玩家登录青川大学综合教务服务平台后，发现自己名下出现了一份从未提交过的“自愿退学申请”，并需要在多个校内系统中取证、检索、复核，找出流程被伪造的原因。

## 在线运行

仓库根目录就是网页目录。部署到 GitHub Pages、Cloudflare Pages 或任意静态站点服务时，入口文件为：

```text
index.html
```

## 本地试玩

可以直接用浏览器打开 `index.html`。如果浏览器限制本地文件访问，可以在仓库根目录运行：

```bash
python3 -m http.server 8765
```

然后访问：

```text
http://127.0.0.1:8765/
```

## 代码结构

- `index.html`：游戏入口
- `game-data.js`：证据、系统模块、旧刊彩蛋和论坛帖等静态数据
- `app.js`：应用总控、统一检索、结局渲染和事件绑定
- `style.css`：界面样式
- `assets/`：游戏内证据图、环境图和门户图
- `scripts/`：规则、状态、动作、视图和测试

## 测试

项目不依赖构建工具。若本机安装了 Node.js，可在仓库根目录运行：

```bash
for f in scripts/*.test.js; do node "$f" || exit 1; done
```
