# 《自愿离校》游戏攻略

对应当前 `code/` / `netlify-drop/` 版本。共 21 条证据；复学前取证夹主要显示 `18`，恢复在籍后会补开后续材料并变为 `21`。

## 总原则

- 验证码可用：`713A` / `0713`。
- 首通建议：先证明“退学流程无效”，恢复在籍后再查 B4、顾天泽和推免链。
- 不想提前进结局，不要点“我已知悉，并自愿离校”。
- 复核页低风险选择：责任口径选“多部门材料互相引用”，材料范围选“只提交流程有效性材料”。
- “无权限 / 已离校”通常不是死局，是提示你先恢复身份或换系统找线索。

## 推荐真结局路线

1. 登录后看教务首页，点流程时间线里的异常节点，拿证据 01。
2. 进材料归档：看退学申请 PDF 信息拿 02；看谈话摘要并输入原始记录相关关键词拿 03；看宿舍清退材料拿 05。
3. 点“撤销申请”触发“账号不存在或已离校”，解锁学工线索。
4. 进学工系统：再次提交异议拿 06；看流转图 / 同源 IP 拿 07。
5. 进邮箱：查看自动转发规则拿 10；顺着归档审计进入归档预览缓存，标记来源字段拿 14。
6. 进家长端缓存：对比家长端和归档文案拿 12；看学生画像拿 13。
7. 进宿舍 / 校园卡 / 图书馆 / 论坛：拿 08、09、11，并记住 `B4`、`0713`、`guest_404`。
8. 进学生事务 / 校园生活 / 信息公开：拿 15、16、18、19。
9. 到终审复核，只提交“流程有效性材料”，先恢复在籍身份。
10. 复学后回心理中心、图书馆、信息公开，补拿 04、17、20、21。
11. 最后回终审复核，提交复学后补充调查材料，触发隐藏真结局“回到名单”。

## 解锁顺序

## 结局速查

| 结局 | 触发方式 | 含义 |
| --- | --- | --- |
| 石沉海底 | 证据很少就去终审复核并提交。 | 申诉被流程吞掉。 |

| 全身而退 | 已能证明流程无效，但混入不完整的敏感真相材料。 | 身份大致保住，真相被封存。 |
| 众目昭彰 | 直接提交大量 B4 / 舆情 / 利益链材料。 | 真相公开，但风险和代价更高。 |
| 拨乱反正 | B4 与利益链证据足够后选择补正路线。 | 推动制度层面纠偏。 |
| 回到名单 | 先低风险复学，再补齐 04、17、20、21 并提交。 | 隐藏真结局：回到保研名单，也把链条带回光下。 |

## 卡关优先查

1. 没方向：看右侧“已出现关键词”，把新词拿去搜索。
2. 没权限：先走终审复核恢复在籍。
3. 邮箱卡住：找自动转发规则，再去归档预览缓存标记来源。
4. B4 卡住：组合搜 `B4 0713 guest_404`。
5. 真结局卡住：确认已经复学，并补齐 04、17、20、21。
